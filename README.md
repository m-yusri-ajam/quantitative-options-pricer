# Quantitative Options Pricer

## 📌 Project Overview
Advanced mathematical modeling for derivative pricing. This project compares the Black-Scholes-Merton model with Binomial Trees to price European and American options.

## 🛠 Features
* **Greeks Calculation:** Real-time tracking of Delta, Gamma, Theta, and Vega.
* **Strategy Simulator:** Models multi-leg strategies (Spreads, Straddles).
* **Early Exercise Logic:** Binomial trees for American-style options.

## 🚀 Impact
Quantifies market risk and provides precise entry/exit pricing for hedged positions.
